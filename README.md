  <h3>TEST TASK for a Backend Engineer position at WinWin.travel</h3>
  <h3>Mini Test: Two Spring Boot Apps with Postgres and Docker</h3>
  <h4 align="left">Goal: Build two small Spring Boot apps (Service A and Service B) that run via docker-compose. 
    Service A handles simple auth with Postgres and exposes a client endpoint that calls Service B. 
    Service B processes the input and returns a result. 
    Service A saves a small record about the processed request.</h4>
